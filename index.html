Face Recognition Attendance System

How to use the scanner:

  1. Allow camera access when prompted
  2. Position your face clearly in the camera view
  3. Click the 'Scan Face' button
  4. The system will check if your face is registered
  5. If recognized, your attendance will be recorded

Registered Attendees

Dashboard Instructions:

  1. Enter the attendee's full name
  2. Select a clear frontal face photo
  3. Click 'Upload Photo' to register the attendee
  4. All registered attendees will appear in the gallery
  5. Use the Scanner tab to check attendance
